Owl Call Center

TaskRouter Call Center with single caller queue

Agent list

Call Center Functionality:

  1. A Twilio phone number is configured with an IVR.
  2. The IVR receives incoming callers, says a message, and put the caller into a TaskRouter managed voice queue.
  3. The voice queue holds incoming callers while they listen to music.
  4. There is a single TaskRouter Workspace:
    1. One Workflow
    2. One TaskQueue
    3. Worker agents
    4. Agent Client call Activities (example, offline, idle, and busy).
  5. There is this Agent application:
    1. agents to set their status activity to idle, so that they can receive calls from callers in the queue (TaskQueue).
    2. This application can either be deployed to Heroku or run on a localhost computer.
    3. The application is a simplified version of the Twilio Call Center course application.
